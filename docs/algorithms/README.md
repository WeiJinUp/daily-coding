## 学习建议

### 刷题路线

1. **基础阶段**：从简单题入手，熟悉语法和基本数据结构
2. **进阶阶段**：中等题为主，掌握常用算法和技巧
3. **提高阶段**：挑战困难题，锻炼思维能力

### 学习方法

- **先思考再看答案**：独立思考 15-30 分钟
- **理解而非记忆**：搞懂原理比记住代码更重要
- **总结归纳**：相同类型的题目要总结规律
- **定期复习**：温故而知新，加深理解
