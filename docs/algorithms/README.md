## 解题技巧

### 常用思维模式

#### 1. 双指针技巧

**滑动窗口（Sliding Window）** 

滑动窗口是处理数组/字符串问题的重要技巧，通过维护一个动态区间来优化时间复杂度（O(n²) → O(n)）。

- **定长滑动窗口**：窗口大小固定，求连续区间的和、平均值等
- **变长滑动窗口**：窗口大小动态，求满足条件的最长/最短子数组

**📜 详细内容**：[滑动窗口完整手册](docs/algorithms/sliding-window)

---

**对撞指针**

两个指针从两端向中间移动，常用于：
- 有序数组的查找问题
- 回文串判断
- 反转问题

---

**快慢指针**

两个指针以不同速度移动，常用于：
- 链表环检测
- 链表中点查找
- 删除倒数第 N 个节点

---

#### 2. 递归与回溯

- **树形问题**：二叉树遍历、路径和
- **排列组合**：全排列、组合总和
- **路径搜索**：迷宫、岛屿问题

---

#### 3. 动态规划

- **状态定义**：明确 dp[i] 的含义
- **状态转移**：找到递推关系
- **优化技巧**：空间压缩、滚动数组

---

#### 4. 贪心算法

- **局部最优**：每步选择当前最优
- **贪心策略**：证明局部最优能导致全局最优
- **正确性证明**：反证法、交换论证

## 学习建议

### 刷题路线

1. **基础阶段**：从简单题入手，熟悉语法和基本数据结构
2. **进阶阶段**：中等题为主，掌握常用算法和技巧
3. **提高阶段**：挑战困难题，锻炼思维能力

### 学习方法

- **先思考再看答案**：独立思考 15-30 分钟
- **理解而非记忆**：搞懂原理比记住代码更重要
- **总结归纳**：相同类型的题目要总结规律
- **定期复习**：温故而知新，加深理解
