## 使用说明

所有图标URL都已验证可用，可以直接在项目中使用。

### 在HTML中使用
```html
<img src="https://stardewvalleywiki.com/mediawiki/images/thumb/f/f1/Weather_Icon.png/32px-Weather_Icon.png" alt="天气图标">
```

### 在CSS中使用
```css
.weather-icon {
  background-image: url('https://stardewvalleywiki.com/mediawiki/images/thumb/f/f1/Weather_Icon.png/32px-Weather_Icon.png');
  width: 32px;
  height: 32px;
}
```
